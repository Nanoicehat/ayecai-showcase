## Why

管理后台当前使用通用的 UI 风格（深色侧边栏 + 靛蓝主色），与前台展示站的清新自然风格（#5b9bd5 蓝 + #7ec8a4 绿 + 圆角卡片 + 柔和阴影）割裂感强。统一视觉语言让整体产品感更好，同时管理后台也能更美观。

## What Changes

将管理后台的视觉风格全面对齐前台 showcase：
- 主色系从靛蓝 #6366f1 → 前台蓝 #5b9bd5，辅色 #7ec8a4
- 侧边栏从深色 #1e293b → 白底/浅灰 + 左侧描边高亮
- 卡片、按钮、输入框等组件圆角和阴影对齐前台设计 token
- 整体排版更加舒适（间距、字体大小对齐前台变量体系）
- 文章列表改为卡片网格布局（类似前台 SolutionsList 的风格）

## Capabilities

### New Capabilities
- `admin-visual-refresh`: 管理后台全局样式 + 组件视觉风格对齐前台

### Modified Capabilities
无

### Removed Capabilities
无
